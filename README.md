# Teample Hub (팀플허브 💛💖)
대학생들의 성공적인 팀 프로젝트(팀플)를 위한 상큼하고 트렌디한 **역할 분배 & 회의 시간 조율 웹 애플리케이션**입니다. 
흰 바탕에 화사한 노랑과 핑크 톤앤매너를 가미한 감각적인 네오브루탈리즘/글래스모피즘 디자인으로 설계되어 눈이 즐겁고 사용하기 편리합니다.
## ✨ 주요 기능
### 1. 👥 팀원 역할 분배 (Role Recommendation)
- '우리 팀원 목록'에 모든 팀원의 이름을 입력해주세요. 또 팀플에 추가로 필요한 역할이 있다면 '역할 분배 보드'에 새 역할을 추가하면 됩니다. 각 역할에 새 할 일을 추가할 수도 있습니다.
- **🎲 역할 자동 분배** 버튼을 누르면, 무작위로 팀원을 매칭하여 역할(조장/자료조사/발표자료제작/발표진행/서기)을 자동 배정합니다.
### 2. 📅 실시간 회의 시간 조율 (Heatmap & Top 3)
- 각 팀원이 자신의 가능한 시간대를 일대일 드래그 앤 드롭 방식으로 간편하게 색칠해 입력할 수 있습니다.
- 입력된 팀원들의 스케줄은 **실시간 히트맵**으로 중첩되어, 겹치는 인원이 많을수록 화사한 그라데이션으로 매칭 확률을 시각화합니다.
- 조율 결과를 분석하여 **가장 많은 조원이 참석할 수 있는 연속 2시간의 회의 시간 Top 3**를 자동 추천합니다.
### 3. 📋 조율 결과 카카오톡 공유
- 매칭된 역할 배정 현황과 추천 회의 시간 목록을 깔끔한 텍스트 리포트로 즉시 조합하여 원클릭으로 클립보드에 복사할 수 있어 단톡방 공유에 최적화되어 있습니다.

## 🛠️ 사용 기술 (Tech Stack)

- **Markup**: HTML5
- **Styling**: Vanilla CSS (CSS Custom Properties, Flexbox, Grid, Glassmorphism & Micro-animations)
- **Logic**: Pure Vanilla JavaScript (Client-side state, LocalStorage)
- **Icons**: Lucide Icons (CDN)
- **Fonts**: Google Fonts (Plus Jakarta Sans, Pretendard)

---

## 📂 리포지토리 파일 목록

- `index.html` (메인 웹페이지 레이아웃 및 뼈대)
- `style.css` (색상 시스템, 애니메이션, 반응형 레이아웃 정의)
- `app.js` (팀원 역량 자동 배정 및 시간표 분석 로직)

---

## 🚀 로컬 실행 방법

이 애플리케이션은 서버가 필요 없는 완전한 **정적 단일 페이지 애플리케이션(SPA)**입니다.
1. 이 저장소(Repository)를 다운로드(Clone)합니다.
2. `index.html` 파일을 더블클릭하거나 크롬/엣지 브라우저 창으로 끌어다 놓으면 즉시 실행됩니다.
3. 웹페이지를 새로고침하더라도 `localStorage`에 입력된 팀원 및 일정 정보가 자동으로 안전하게 보존됩니다.
